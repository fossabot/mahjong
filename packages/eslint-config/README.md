# `@mahjong/eslint-config`

## Usage

For Next App.

```json
{
  "extends": ["@mahjong"]
}
```

## Presets

### `@mahjong/eslint-config/base`

For Node.js or TypeScript.

### `@mahjong/eslint-config/nest`

For NestJS app/package within `+jest`.

## Addon

### `@mahjong/eslint-config/+jest`

For Jest.
